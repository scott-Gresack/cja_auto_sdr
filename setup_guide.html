üü¢ Step 1: Install Node.js

Option A: Using Homebrew (Recommended)

First, install Homebrew if not already installed:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Then install Node.js:

Verify installation:

Option B: Direct Download

Download from nodejs.org (LTS version 18 or later) and follow the installer.

üìà Step 4: Setup CJA SDR Generator

Navigate to Project

cd /Users/gresack/ai_mar/cja_auto_sdr_original

Create Virtual Environment

python3 -m venv venv source venv/bin/activate

Install Dependencies

pip install -r requirements.txt

Configure API Credentials

Create a config.json file (see config.json.example):

{ "acp": { "org_id": "your-org-id@AdobeOrg", "tech_account_id": "your-tech-account-id", "api_key": "your-api-key", "secret": "your-secret", "ims_host": "https://ims-na1.adobelogin.com" }, "cja": { "company_id": "your-company-id", "api_token": "your-api-token" } }

Test the CLI

python cja_sdr_generator.py --help

This shows all available commands and options.

Generate Your First SDR

python cja_sdr_generator.py --data-view-id YOUR_DATA_VIEW_ID --output results.xlsx
‚úì Complete! CJA SDR Generator is ready. Run it whenever you need to document your data views.

üîß Troubleshooting

Problem: "npm: command not found"

Node.js might not be in your PATH. Try:

# Find where Node is installed which node # Or reinstall brew uninstall node brew install node@18

Problem: "ModuleNotFoundError" in Python

Ensure your virtual environment is activated:

cd /Users/gresack/ai_mar/cja_auto_sdr_original source venv/bin/activate pip install -r requirements.txt

Problem: API Authentication Fails

Check your credentials are correctly set in environment variables. For Next.js:

cat .env.local # Verify contents

Problem: Port 3000 Already in Use

Run on a different port:

npm run dev -- -p 3001
