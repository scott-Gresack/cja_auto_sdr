Common patterns and request examples.

Fetching Batches with Configuration

JavaScript/TypeScript import { fetchWithConfig } from '@/lib/fetch-with-config'; // Automatic configuration injection const batches = await fetchWithConfig( '/api/batches?datasetId=MY_DATASET&limit=25' ); const data = await batches.json(); console.log(`Retrieved ${data.length} batches`);

Creating AEP Configuration

Configuration Creation const config = { clientId: 'YOUR_CLIENT_ID', clientSecret: 'YOUR_CLIENT_SECRET', orgId: 'YOUR_ORG_ID', sandbox: 'prod', sandboxId: 'YOUR_SANDBOX_ID', isActive: true }; // Save to IndexedDB await saveConfig(config);

Token Validation

Validate OAuth Token const response = await fetch('/api/oauth/aep-token', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ token: ACCESS_TOKEN }) }); const { valid, expiration } = await response.json(); console.log(`Token valid: ${valid}, Expires: ${expiration}`);

Segment Monitoring

Fetch Segment Jobs // Get recent segment jobs const jobsResponse = await fetchWithConfig('/api/segmentation/segment-jobs?limit=50'); const jobs = await jobsResponse.json(); // Filter by status const activeJobs = jobs.filter(job => job.status === 'RUNNING'); console.log(`Active jobs: ${activeJobs.length}`);
